Data & Code

Publicly available datasets and analysis tools

I believe in open science. Below are datasets and code associated with my published work, along with tools I have developed for planetary data analysis. If you use any of these in your research, a citation to the associated publication is appreciated.

Datasets

Titan Equatorial Spectral Clusters

Geospatial-topological clustering results from Cassini/VIMS observations of Titan's equatorial region. Associated with Kutsop et al. (2024), Icarus.
